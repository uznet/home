# \<var> tag
## Purpose 
### 변수를  선언합니다.

## Syntax
<var\
expr = "ECMAScript_Expression"\
name = "string"\
/>





## Attributes
|Name |Description |Required |Default|
|-----|------------|---------|-------|
|name |변수이름 .|true| |    
|expr |ECMA 표현식입니다.|false|undefined |
