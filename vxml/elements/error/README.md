# \<error> tag
# Purpose 
## 플랫폼 또는 \<throw../>  요소에 의해 throw 된 'error' 이벤트 를 처리합니다.

# Syntax
\<error
cond = "ECMAScript_Expression"
count = "Integer"/>



# Attributes
|Name |Description |Required |Default|
|-----|------------|---------|-------|
|cond |\<catch> 참조 |     |      |
|count|\<catch> 참조 | | |
