# \<foreach> tag
## Purpose 
### ECMAScript 항목 배열을 반복합니다.

## Syntax
\<foreach\
array = "ECMAScript Array Expression"\
item = "variable"/>



## Attributes
|Name |Description |Required |Default|
|-----|------------|---------|-------|
|array |Array 타입 ECMA Script object | true    |      |
|item |루프의 각 반복시 각 배열 항목을 저장하는 변수입니다.|true|  |
