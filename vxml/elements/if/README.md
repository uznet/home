# \<if>\<else>\<elseif>   tag
## Purpose 
### 조건부 논리를 수행합니다
## Syntax
\<if cond="ECMAScript_Expression"> \
   \<!-- executable content --> \
\<elseif cond="ECMAScript_Expression"/> \
   \<!-- executable content --> \
\<elseif cond="ECMAScript_Expression"/> \
   \<!-- executable content --> \
\<else/> \
\<!-- executable content --> \
\</if>



## Attributes
|Name |Description |Required |Default|
|-----|------------|---------|-------|
|cond |true 또는 false로 평가되는 ECMAScript 식입니다 |   true  |      |
