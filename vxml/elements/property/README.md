# \<property> tag
## Purpose 
### VocieXML platform 의 설정값을 지정
## Syntax
\<property\
expr = "ECMAScript_Expression"\
name = "string"\
value = "string"\
/> 






## Attributes
|Name |Description |Required |Default|
|-----|------------|---------|-------|
|name |속성이름.| true    |    |
|value|속성값|false||    
|expr|ECMAScript 형식의 속성값 |false||
