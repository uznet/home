# \<log> tag
## Purpose 
### 디버그 메시지 생성

## Syntax
\<log\
label = "string"\
expr = "ECMAScript_Expression"\
/>



## Attributes
|Name |Description |Required |Default|
|-----|------------|---------|-------|
|label|로그 메시지를 분류하는 데 사용할 수있는 문자열입니다 | false    |      |
|expr |String 타입의  ECMAScript 표현식.|false| undefined|
