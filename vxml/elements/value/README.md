# \<value> tag
# Purpose 
## \<value> 요소는 표현식의 값을 프롬프트에 삽입하는 데 사용됩니다.

# Syntax
\<value expr = "ECMAScript_Expression" />




# Attributes
|Name |Description |Required |Default|
|-----|------------|---------|-------|
|expr |렌더링 할 ECMA 표현식입니다.|true| |
