# \<disconnect> tag
## Purpose 
### 세션 연결 끊기

## Syntax
\<disconnect\
namelist = "string"/>



## Attributes
|Name |Description |Required |Default|
|-----|------------|---------|-------|
|namelist|인터프리터 컨텍스트에 반환되는 공백으로 분리된 변수 목록 이름들|false||
